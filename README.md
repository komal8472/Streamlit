

This project is an interactive Statistical Calculator built using Streamlit that allows users to compute basic statistical measures from user-entered numerical data.
The application is designed to make statistics simple, visual, and easy to understand for beginners.

---

## Objective

To calculate key statistical measures interactively

To allow users to input custom numerical data

To display statistical results with clear explanations and formulas

---

## Features

Accepts comma-separated numerical input from the user

Automatically sorts the entered values

Computes important statistical measures:

    Mean

    Median

    Mode

    Variance

    Standard Deviation

Displays mathematical formulas for better conceptual understanding

Clean and user-friendly web interface using Streamlit

---

## Tools

Python

NumPy

Streamlit

Collections module

---

## How the Application Works

User enters numerical values separated by commas

Application processes the input data

Statistical measures are calculated in real time

Results are displayed clearly along with formulas

---

## Learning Outcomes

Understanding how statistical measures are computed programmatically

Hands-on experience with Streamlit for data applications

Practical implementation of NumPy operations

Improved clarity of statistical concepts through visualization

---

## How to Run

Install required libraries

Open terminal in the project folder

Run the Streamlit application 
use this - streamlit run file_name.py

Enter numeric values separated by commas

View instant statistical results on the browser
